Don't know how to use the terminal? Don't worry!




about command




The about command allows you to extract key information about me. Type "about" followed by a space and a parameter to obtain information. Parameters are listed below.

Parameter Name Information
age The age of Ethan E.
name This one is sort of obvious. What is Ethan's name?
residence Where does Ethan live?
school Where does Ethan study?
email At what email address may I contact Ethan?
birth When was Ethan born?




bio command
